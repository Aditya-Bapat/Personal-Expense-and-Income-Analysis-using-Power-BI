This project uses Power BI to analyze personal transaction data with the aim of understanding income, expenses, and overall spending patterns. The dashboard provides a clear view of how finances change over time and highlights major expense categories and high-spending periods.

The focus of this project is on basic data analysis and visualisation, keeping the insights simple and easy to interpret.

**Business Objective**
To track income and expenses over time, identify major spending areas, and compare monthly spending patterns to support better financial awareness and decision-making.

**Dataset**
The dataset contains transaction-level data with the following key fields:
1)Date
2)Transaction Type (Credit / Debit)
3)Category
4)Amount
The data is used for learning and analysis purposes.

**Dashboard Features**
1) Monthly Income vs Expense Trend
Visualizes how income and expenses change over time.
2) Expense Breakdown by Category
Highlights major spending areas such as rent, groceries, utilities, and shopping.
3) High-Spending Months
Compares total expenses across months to identify periods with higher spending.
4) Detailed Transaction Table
Provides transaction-level details for deeper analysis.
